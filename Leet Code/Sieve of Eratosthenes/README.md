# Problem
Find all prime numbers in the first `n` natural numbers

### Input
- `n`: Number of integers

### Output
- A list of prime integers in the first `n` natural numbers

##### Example 1:
```
Input: 10
Output: [2, 3, 5, 7]
```

##### Example 2:
```
Input: 25
Output: [2, 3, 5, 7, 11, 13, 17, 19, 23]
```
